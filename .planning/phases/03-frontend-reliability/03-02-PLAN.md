<objective>
Implement VARCHAR/CHAR truncation detection and inline warning (FRONT-02).

Purpose: When a user edits a cell containing a VARCHAR/CHAR-typed column and enters a value longer than the column's defined maximum length, the existing `enforceTypeConstraints` silently truncates the value. This fix surfaces the truncation via an inline tooltip showing the original and max lengths, so users are aware their input was modified.

Output: Modified sqlStatements.ts (return type), useBidirectionalSync.ts (caller updated), sqlStatements.test.ts (tests updated), useInlineEdit.ts (truncation callback), TableSection.vue (tooltip rendering)
</objective>

    ```
    export const enforceTypeConstraints = (
      val: any,
      rawType: string
    ): { value: any; truncated?: { originalLength: number; maxLength: number } } => {
      if (val === null || val === undefined) return { value: val }
      const strVal = String(val)
      const typeUpper = parsePrimaryType(rawType).toUpperCase()
      const charMatch = typeUpper.match(/(?:VARCHAR|CHAR)\s*\((\d+)\)/)
      if (charMatch) {
        const maxLength = parseInt(charMatch[1], 10)
        if (strVal.length > maxLength) {
          return {
            value: strVal.substring(0, maxLength),
            truncated: { originalLength: strVal.length, maxLength }
          }
        }
      }
      return { value: val }
    }
    ```
